Rocket League Stats Bot

Inspired by Minecraft PaceMan sessions bots, RL_StatsBot allows you to follow your session, rank and other cool stuff for Rocket League!

Currently, the bot is in private beta. You can use it in participating channels, link your account, but your sessions won't be accurate.
When the bot becomes public, you will be free to add it to your Twitch channel!

Sessions are only tracked when you are live and playing Rocket League, or turned on offline mode. More on that below.

Commands

The default command prefix is *. Each chatroom will be able to change their prefix soon™.

Rocket League

Command Description Examples
*link Link your Rocket League account to the bot.
You can only link one account at a time.

Optional parameter:
--platform or -p <epic|steam|psn|xbl|switch> (defaults to epic)

Steam requires the numeric ID. But might as well link you Epic account that Rocket League uses, even if you play from Steam. 		*link MyEpicUsername

These two do the same thing:
*link --platfrom psn PlayStationUsername
*link -p psn PlayStationUsername

*link -p xbl XBoxAccountName
*link -p steam 123456789
*link -p switch SwitchPlayerWow
*unlink Unlink your account from the bot

If you want to link to another account, you can use the *link command directly, and it will override your previous association. 		*unlink
*session
!session		 Get current session information.

Without parameters:
  • *session defaults to yourself if you linked your account, otherwise the current streamer.
  • !session defaults to the streamer, or yourself if the streamer has not linked their account.
Note: !session is only active when the stream category is set to Rocket League.

Optional parameters:
  • --platform or -p <epic|steam|psn|xbl|switch> (defaults to epic), works the same as the *link command.
  • username or @username.
When using username, you look up the session of the given username on the optionally given --platform.
When using @username, you look up the session of the mentioned Twitch user, if they have an account linked. --platform is ignored. 		!session
*session
*session EpicUsername
*session -p xbl XBoxAccount
*session @TwitchUsername
*rank
!rank		 Get a user's ranks.

Optional parameters:
  • --platform or -p <epic|steam|psn|xbl|switch> (defaults to epic), works the same as the *link command.
  • username or @username.
Works the same as the *session command, but returns rank info instead. 		!rank
*rank
*rank EpicUsername
*rank -p xbl XBoxAccount
*rank @TwitchUsername
*start-session
*stop-session		 Starts or stops an offline session.

If you are live and on the Rocket League category, you do not need these commands.

Only available to whitelisted users while in private beta. 		*start-session
*stop-session
